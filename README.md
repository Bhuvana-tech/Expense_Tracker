# Budget Tracker

## About

This is a simple budget tracker web app I built to practice basic web development. It allows users to set a monthly budget, add expenses, and see how much money is left.

## Features

* Set a budget limit
* Add expenses with name and amount
* View total spending
* See remaining balance
* Delete expenses

## How it works

First, you enter your budget. Then you can keep adding expenses. The app updates the total spent and remaining amount automatically using JavaScript.

## Tech used

* HTML for structure
* CSS for styling
* JavaScript for functionality

## How to run

Just open the `index1.html` file in your browser.
